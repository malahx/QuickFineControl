#### What is it?

QuickFineControl is a plugin which adds the possibility to toggle the precision control at the load of a vessel.

#### How to install it?

Unzip all files. Put the QuickFineControl folder in your KSP/GameData folder.

#### How to update it?

Unzip all files. Merge the new QuickFineControl folder with the old folder which is in your KSP/GameData folder.

#### How to uninstall it?

Delete the QuickFineControl folder in your KSP/GameData folder.
